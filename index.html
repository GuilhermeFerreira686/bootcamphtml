    • FUNDAMENTOS DO HTML

    Voltar

    Para começo de conversa, é importante frisar que o htmlNÃO é uma linguagem de programação, mas sim uma linguagem de marcação de texto, basicamente é a estrutura da sua página de site.Pelo HTML podemos criar sites com diversas páginas, links e formas variadas de produzir um texto, utilizando fontes, cores ou larguras e espaçamentos diferentes, e para construir isso tudo, usamos as famosas TAGS. Todas essas informações são programadas e enviadas até o Cliente, resumidamente é a estrutura de texto que você vê quando entra em qualquer site. A estrutura do HTML é composta pela HEAD, tudo aquilo que se encontra na ponta do site, lá em cima. Pelo BODY, onde se encontra o conteúdo da sua produção de texto, todos os parágrafos, títulos, subtítulos e assim por diante e por último o FOOTER, onde vão ser localizadas infomações ao final da página.

    TAGS

    Voltar

    Acerca das TAGS, podemos dizer que as tags são as ferramentas de trabalho do HTML, com as tags podemos colocar funções na nossa estrutura de texto e criar os títulos, adicionar imagens, ou seja, sem as tags, não existe html. Além disso, podemos ancorar links, adicionar comandos CSS, entre outras coisas, alguns exemplos de TAGS são: "strong" que faz o texto ficar em negrito e o "i" onde as palavras mudam para a fonte itálica, DESSA FORMA, mas além dessas existem diversas tags que podemos usar na criação de estruturas.

    Alguns outros exemplos de TAGS:

    • basefont
    • target
    • title
    • style
    • class
    • img
    • u

      • LINKS E SITES

      Voltar

      Sobre os links, para adicionar links em nosso site, usamos as tags "link href" assim podemos adicionar a URL do site em que, quando o usuário clicar por cima de um site, ele seja direcionado a outra página. Também podemos utilizar o "a" de âncora, para redireionar o usuário a títulos específicos dentro da própria página, além disso, temos o "self" que abre outro site na mesma aba do navegador, o "_blank" que abre um site em outro aba e também o "title" que quando adicionado ao lado de um título adiciona uma mensagem ao deslizar a seta do mouse por cima de uma palavra específica. Não podemos esquecer da "Target" que serve de atributo para diversas TAGS.

      LISTAS ORDENADAS E NÃO ORDENADAS

      Voltar

      Para finalizar, as listas ordenadas e não ordenadas são utilizadas quando necessita de um espaço para guardar informações ou apenas listas algum tipo de conteúdo, para as listas ordenadas, o nome literal já diz, utilizamos uma lista com número, para isso, usa-se a TAG "ol" seguido dos itens da lista "li" (List item) e para as listas não ordenadas, com marcadores, usamos a TAG "ul" seguida também pelos itens da lista "li".
